Coordinación de Informática – Acapulco, Guerrero, México.

Tareas realizadas: Creación de un sistema de control de inventario de TIC’s para la delegación IMSS Guerrero con el objetivo de agilizar los reportes de inventario con que contaba la delegación.

  • Se utilizó la metodología Scrum y las pruebas fueron realizadas cada semana mediante la red local por el coordinador de informática.
  • El proyecto se terminó en junio de 2019 y fue implementado en la red de la delegación.
